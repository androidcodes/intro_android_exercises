Open HelloWorldDemo in Eclipse
Create a second activity if one doesn't exist
Create a button in the first activity
When the button is clicked, use an explicit intent to open the second activity
Pass a string with key text in intent bundle for the second activity
Display that string value in a TextView within the second activity

## Snippets

// In Activity
Button btnLaunchSecond = (Button) findViewById(R.id.btnLaunchSecond);
btnLaunchSecond.setOnClickListener(new OnClickListener() {
  @Override
  public void onClick(View v) {
    Intent i = new Intent(ExplicitIntentActivity.this, SimpleBundleDemoActivity.class);
    i.putExtra("text", "Passed String Extra!");
    startActivity(i);
  }
});

// In Launched Activity
String initialText = getIntent().getStringExtra("text");
TextView tvDisplayText = (TextView) findViewById(R.id.tvDisplayText);
tvDisplayText.setText(initialText);
